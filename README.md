# Job Portal Web Application

A full-stack job portal built using Spring Boot, Spring Security, Thymeleaf, and MySQL.  
The platform allows recruiters to post jobs and candidates to search and apply for opportunities.
This app was my learning, used as an open source

## 🚀 Features

- User registration & login
- Recruiter job posting
- Candidate job search
- Resume upload
- Saved jobs
- Global search

## 🛠 Tech Stack

- Java 17
- Spring Boot
- Spring Security
- Spring Data JPA (Hibernate)
- Thymeleaf
- MySQL
- Maven
